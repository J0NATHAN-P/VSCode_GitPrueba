Ejercicio del día 9

Elaborar por: Jonathan David Telule Pinti

Indicaciones: La ESFE desea desarrollar un BOT automatizado de atención al cliente, para gestionar las consultas más frecuentes que se realizan en las redes sociales

El BOT debe de mostrar los siguientes elementos:
  • Un Saludo
  • Espacio para digitar el nombre por medio de un INPUT, este debe de llevar un ID
  • Espacio para digitar el año de nacimiento por medio de un INPUT, este debe de llevar un ID
  • Un Botón para enviar los datos, este debe de contener el evento 'onclick'
  • Establecer una etiqueta DIV con el ID="divRespuesta" para mostrar el mensaje procesado del BOT

Las acciones del BOT deben ser las siguientes mediante una función de JavaScript:
  • Obtener los datos de los campos INPUT
  • Calcular la edad (Restar el año actual 2022 menos el año de nacimiento)
  • Personalizar un mensaje con el nombre y edad de la persona
  • Mostrar la opciones que los usuarios tienen disponible, las cuales son:
    • Información sobre las carreras
    • Costos de Inscripción y matricula
    • Fechas de los cursos propedéuticos
    • Información de la rifa Agape
